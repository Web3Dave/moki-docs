# Introduction

Welcome to the **Moki Network** documentation.

## What is Moki?

Moki is a Web3 chat platform built on decentralized principles, featuring:

- **Decentralized Service Nodes**: Distributed infrastructure for resilient messaging
- **Merkle Tree Audit Chains**: Cryptographic proof of message integrity
- **Economic Incentives**: Proof of stake and availability rewards for service providers

## Getting Started

To start using Moki, you'll need to:

1. **Register an account** on the Moki Network
2. **Connect to a service node** for messaging capabilities
3. **Understand the JSON-RPC API** for integration
