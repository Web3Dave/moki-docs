# Moki Documentation

Official documentation for Moki Network - a Web3 chat platform with decentralized service nodes and merkle tree-based audit chains.

This website is built using [Docusaurus](https://docusaurus.io/).

## Installation

```bash
yarn
```

## Local Development

```bash
yarn start
```

This command starts a local development server and opens up a browser window. Most changes are reflected live without having to restart the server.

## Build

```bash
yarn build
```

This command generates static content into the `build` directory.

## Deployment

This repository automatically deploys to GitHub Pages via GitHub Actions on every push to the `main` branch.
